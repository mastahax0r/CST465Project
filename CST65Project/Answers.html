Q: Explain why the alt attribute of the img tag is important.
A: The alt attribute of the img tag is important for two reasons. First, and most important, it enables greater accessibility of the web for people with disabilities that prevent them from seeing the page directly (ie. The use of screen readers for the blind). Secondly, old browsers and bandwidth constrained visitors may not be capable of loading images.
Q: What is the purpose of name mangling?
A: Name mangling of control ID’s helps ensure every control on the page has a unique ID, even if the same kind of control is used multiple times.
Q: Explain the difference between redirects that use a 301 HTTP status code and ones that use a 302 HTTP status code.
A: A 301 error code means the resource has moved permanently, whereas a 302 error means the resource has moved only temporarily. This is an important distinction when the page is being retrieved though an automated processes, such as a web crawler.
